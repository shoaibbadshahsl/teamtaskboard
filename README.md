# This is Node + Express Project 
- It's a complete project for managing team tasks
- All the API's for creating tasks 
- Updating tasks
- Assign user to the task
- status of the tasks
- list all the users
- list all the stats for the tasks in the dashboard page
- Auth (login and register)
- complete flow

## Getting Started

- First clone the project (repo)
```sh
git clone repo link
```

- change the directory 
```sh
cd TEAMTASKBOARD
```

- Run the
 ```sh 
npm install
```

- Then finally run the npm run dev 
Server should run automatically when starting a workspace. To run manually, run:
```sh
npm run dev
```

## To test if everything is running fine 
- Enter the url in your browser

```sh
http://localhost:5000/api/health
```
